Googleフォーム連携 — 登録データの取り込み

参加者の登録内容と診断結果は、Googleフォームへ自動送信されます（assets/js/config.js で設定）。 フォームの回答スプレッドシートから「ファイル → ダウンロード → CSV」で保存し、ここで取り込んでください。

セットアップ手順（初回のみ）
  1. Googleフォームを作成し、記述式の質問を10個追加：名前 / ニックネーム / 年齢 / 年代 / 性別 / 所属教会 / メールアドレス / Discord ID / Mission Type / サブタイプ
  2. フォーム右上の「⋮ → 事前入力したリンクを取得」で全質問にダミー値を入れてリンクを発行し、URL内の entry.XXXXXXX 番号を控える
  3. assets/js/config.js の action（フォームURLの /viewform を /formResponse に置換）と各 entries に貼り付ける
  4. 以後、参加者が診断を終えるたびに回答が自動でフォームに集まります。CSVの列名は上記の質問名のままでOK
